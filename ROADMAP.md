# HyaloOS Roadmap
Autor Patryk "MijagiKutasamoto" Szkudarek

## Version 0.2.0

- Rozbudowa centrum ustawien o backend ustawien, aby zmiany byly trwale i obslugiwaly realne komponenty sesji.
- Dzialajace ustawienia wygladu, ktore zmieniaja sie konsekwentnie w calym srodowisku graficznym HyaloOS.
- Domkniecie centrum ustawien tak, aby kluczowe opcje wygladu i zachowania DE byly dostepne i faktycznie podlaczone.
- Dodanie autostartu aplikacji i procesow sesyjnych.
- Dodanie konfiguracji i obslugi skrotow klawiszowych.
- Przebudowa terminala na styl pasujacy wizualnie i UX-owo do reszty DE.

## Version 0.3.0

- Quick panel z szybkimi akcjami zintegrowany z panelem.
- Aplikacja Obrazki jako natywna przegladarka grafik dla HyaloOS.
- Manager plikow dopasowany wygladem i workflowem do reszty srodowiska.

## Cross-Cutting Tracks

- Replace remaining generic system icons in panel utilities and quick controls with HyaloOS-specific symbolic variants where it improves recognition without reducing clarity.
- Add a fuller symbolic icon set for settings categories and launcher filters so the control center and launcher feel visually consistent.
- Review compositor/session surfaces and app windows for any remaining icon-name mismatches between desktop entries, GTK application IDs, and panel resolution.
